# v16.1.0 — Talk Interaction State Prototype

## Purpose

Adds the first local-state talk UI prototype.

## New route

```text
/talk-interaction
```

## What it does

- lets the user select explanation cards
- updates the visible answer locally in the browser
- demonstrates a conversational product guide

## What it does not do

- no microphone
- no recording
- no provider call
- no external AI call
- no database write
- no billing
- no checkout
- no webhook
