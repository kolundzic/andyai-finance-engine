# v14.8.0 — Talk Safety Boundary Seal

## Locked boundary

The talk layer may:

- explain product purpose
- explain proof
- explain routes
- explain blocked actions
- prepare client-friendly narrative

The talk layer may not:

- record audio
- activate microphone
- call external AI providers
- synthesize speech
- write database records
- bypass approval
- activate billing
- activate checkout
- execute financial actions
