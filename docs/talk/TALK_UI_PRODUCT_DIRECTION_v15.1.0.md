# v15.1.0 — Talk UI Product Direction

## Goal

Create a product guide layer that feels like the engine can talk to a client, while remaining fully static and safe.

## UX promise

The page should answer:

- What is this product?
- What proof exists?
- What is blocked?
- Why does human approval matter?
- What is the next client step?

## Boundary

No microphone.

No recording.

No provider.

No database write.

No payment action.
