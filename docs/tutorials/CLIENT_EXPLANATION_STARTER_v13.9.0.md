# Client Explanation Starter

## One-minute version

AndyAI Finance Engine helps turn business plans and source documents into structured financial model workflows.

The first public version proves the safety boundary:

- the system can show status
- the system can show proof
- the system cannot write data from public routes
- human approval stays above automation

## Business value

A client does not only need fast financial output.

A client needs output that can be reviewed, explained, and trusted.
