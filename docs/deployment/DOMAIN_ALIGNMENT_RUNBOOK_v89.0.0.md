# Domain Alignment Runbook

## Target domain

finance.andyai.ai

## Manual Vercel and DNS steps

1. In Vercel, add finance.andyai.ai to the project domains.
2. In GoDaddy DNS, create or verify the required CNAME record.
3. Wait for DNS propagation.
4. Redeploy latest main.
5. Confirm the canonical URL in public copy.
