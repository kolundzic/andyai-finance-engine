# MASTER TOC v152-v160 — Engine Schema and Architecture Road

## Meaning

This road starts the real machine layer after v151 opened the Engine Completion Road.

## Srpski

Ovaj put počinje stvarni sloj mašine nakon što je v151 otvorio Engine Completion Road.

## v152.0.0 — Finance Workflow Schema Master

Defines the core financial workflow schema: input, task, agent step, review, approval, output.

## v153.0.0 — Assumptions Ledger Data Model Master

Defines assumptions, sources, confidence, risk, review status, and approval linkage.

## v154.0.0 — Evidence Pack Data Model Master

Defines evidence pack structure: source, claim, assumption, risk, approval, output reference.

## v155.0.0 — Human Approval State Machine Master

Defines approval states and transitions for governed finance output.

## v156.0.0 — Pilot Request Lifecycle Master

Defines pilot request lifecycle from intake to qualified pilot conversation.

## v157.0.0 — Model Preview Data Contract Master

Defines model preview contract: projections, assumptions, risk notes, evidence references, readiness.

## v158.0.0 — Risk and Missing Data Rules Master

Defines risk and missing-data rules that block unsafe or incomplete financial output.

## v159.0.0 — Export Package Contract Master

Defines export package contract: PDF, JSON, Markdown memo, evidence appendix, client handoff.

## v160.0.0 — Engine Architecture Lock Master

Locks the first engine architecture baseline after schema and contract layers.
