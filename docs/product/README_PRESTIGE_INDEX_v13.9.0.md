# AndyAI Finance Engine — Prestige Product Index

## What it is

AndyAI Finance Engine is a governed financial modeling foundation.

It turns business inputs into structured financial model workflows with:

- read-only public proof
- audit trails
- approval gates
- Supabase-backed memory foundation
- controlled runtime boundaries

## What it is not

It is not a live payment system yet.

It is not a write-runtime automation yet.

It does not expose service role keys.

It does not mutate database records from public routes.
