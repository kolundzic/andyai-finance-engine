# TAPFORGE Supabase Lab Setup Evidence Note v9.1.0

## Purpose

This note defines what TAPFORGE may later record about Supabase Lab setup.

## Evidence Fields

TAPFORGE may record:

- project label
- setup operator
- setup date
- repo tag used
- env guard result
- preflight result
- migration status
- RLS review status
- operator approval status

## Forbidden Evidence

TAPFORGE must not record:

- service role key
- database password
- raw secret values
- screenshots containing secrets
- unrestricted connection strings

## Rule

Evidence proves the setup happened.

Evidence must not expose the system.
