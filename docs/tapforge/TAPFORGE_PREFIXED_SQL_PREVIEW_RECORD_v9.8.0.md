# TAPFORGE Prefixed SQL Preview Record v9.8.0

## Purpose

This document defines what TAPFORGE must record about the prefixed SQL preview.

## Evidence Fields

- source repo
- source tag
- preview bundle path
- prefix decision version
- generated at
- operator
- review state
- notes

## Required Result

Allowed values:

- blocked
- needs_revision
- prefixed_preview_accepted_for_lab_apply_candidate

## Rule

TAPFORGE may record preview acceptance.

TAPFORGE may not treat preview acceptance as SQL apply approval.
