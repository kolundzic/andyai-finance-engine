# v8.1.0 — Supabase Free Account Deployment Strategy

## Summary

v8.1.0 adds the Supabase Free Account Deployment Strategy.

## Scope

- Defines Free account as lab/proof environment
- Defines Finance Engine as source-of-truth for finance schema
- Defines TAPFORGE as orchestrator, not duplicate schema owner
- Defines namespace and prefix strategy
- Defines data budget strategy
- Defines upgrade rule toward Pro/production
- Confirms no live Supabase setup is performed

## Safety

No Supabase project is created.

No migration is applied.

No secret is added.

No Vercel environment variable is changed.

No checkout, webhook, billing provider, subscription enforcement, or paid runtime behavior is introduced.
