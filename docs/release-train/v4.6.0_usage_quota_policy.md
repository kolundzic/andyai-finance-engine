# v4.6.0 — Usage Quota Policy

## Summary

This release adds the first usage quota policy for AndyAI Finance Engine.

## Scope

- Project quota direction
- Model draft quota direction
- Export quota direction
- Team member quota direction
- Audit retention direction
- Manual review direction
- Quota safety rule

## Safety

No runtime quota enforcement, billing provider, checkout route, webhook, metering table, or subscription logic is introduced.

## Release Mode

Local-only until the full v5.0.0 Monetization Gate line is complete.
