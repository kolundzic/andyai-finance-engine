# v9.2.0 — Local .env Setup Assistant

## Summary

v9.2.0 adds a local assistant script for creating `.env.local` safely.

## Scope

- Adds `scripts/create-supabase-env-local.sh`
- Adds `.env.local` and `.env*.local` to `.gitignore`
- Adds local env setup documentation
- Adds operator checklist
- Verifies existing env guard and live preflight in non-strict mode

## Safety

No real secret value is added.

No Supabase project is created automatically.

No migration is applied.

No Vercel environment variable is changed.

No checkout, webhook, billing provider, subscription enforcement, or paid runtime behavior is introduced.
