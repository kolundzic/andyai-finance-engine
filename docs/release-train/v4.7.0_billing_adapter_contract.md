# v4.7.0 — Billing Adapter Contract

## Summary

This release adds the first billing adapter contract for AndyAI Finance Engine.

## Scope

- Provider-neutral billing direction
- Future provider candidates
- Billing event types
- Internal state mapping direction
- Manual invoice mode
- No billing mode
- Billing safety rules

## Safety

No checkout route, webhook, provider SDK, secret key, payment form, subscription enforcement, or live billing behavior is introduced.

## Release Mode

Local-only until the full v5.0.0 Monetization Gate line is complete.
