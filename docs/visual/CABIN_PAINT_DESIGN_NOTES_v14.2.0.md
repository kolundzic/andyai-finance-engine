# v14.2.0 — Cabin Paint Design Notes

## Visual direction

The cabin should feel:

- serious
- calm
- premium
- financial
- evidence-driven
- human-controlled

## Design improvements for next UI rounds

| Area | Improvement |
|---|---|
| Navigation | Keep fewer main items visible on small screens |
| Cards | Use evidence-style cards with short proof labels |
| Hero | One strong headline, one clear proof promise |
| Status | Prefer blocked/verified/read-only language |
| Client routes | Make product routes look intentional, not accidental |
| Talk layer | Make it feel like a guided assistant, not a chatbot toy |

## Rule

Beautiful, but verifiable.
