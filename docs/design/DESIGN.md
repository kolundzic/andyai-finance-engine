# DESIGN.md — AndyAI Finance Engine

## Purpose

This file defines the first practical design direction for AndyAI Finance Engine.

The goal is not luxury decoration. The goal is immediate product understanding.

## Core UI language

- dark professional finance background
- clear white headings
- emerald for action / safe progress
- cyan for proof / workflow
- amber for caution / risk / approval
- rounded cards
- visible step numbers
- simple CTA buttons

## Practical rule

Every page must answer:

1. What is this?
2. What does the user do?
3. What does the engine do?
4. What does the human approve?
5. What proof remains?
6. What is the next action?
