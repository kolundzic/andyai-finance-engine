# VISUAL_CANON.md — AndyAI Finance Engine

## Visual style

AndyAI Finance Engine should look like a controlled finance workflow system, not like a generic AI landing page.

## Emotional tone

- calm
- precise
- trustworthy
- practical
- professional
- understandable

## Core symbols

- ledger / evidence trail
- approval gate
- model preview
- workflow arrows
- human review checkpoint
- pilot journey

## First five required visuals

1. Finance Workflow: User → Agent → Proof → Approval → Output
2. Human Approval Gate
3. Financial Agent vs Excel
4. Trust Ledger / Evidence Flow
5. Pilot Customer Journey
