# UI_PATTERNS.md — AndyAI Finance Engine

## Required UI patterns

### Hero section
Must state product value in one sentence.

### Proof strip
Must show model preview, assumptions, approval, and evidence.

### Workflow cards
Must show step-by-step sequence.

### CTA block
Must always give the user a next action.

### Trust block
Must explain what remains blocked and why blind finance automation is not allowed.

## Rule

No page should be only text. Every public page needs a visible structure.
