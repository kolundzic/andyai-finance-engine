# Beginner Glossary

## Runtime
The part of the system that actually does something while the product is running.

## Static page
A page that shows information but does not submit, write, or execute anything.

## Provider
An external AI service, such as a text, voice, or model API.

## Database write
A real action that stores or changes data in a database.

## Service role key
A powerful backend secret key. It must never appear in public client code.

## Human approval gate
A rule that a human must review and approve before a serious action is allowed.

## Evidence pack
A structured set of proof, assumptions, warnings, and review notes that travels with a financial model.

## TAP-TAP
The AndyAI workflow: apply script, verify, commit, tag, push, and lock.
