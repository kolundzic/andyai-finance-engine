# WORKFLOW

```text
1. Business brief / source documents
2. Data extraction
3. Assumption separation
4. Financial model build
5. Verification checks
6. Human review and approval
7. Export pack
```

## Initial demo workflow

- Upload or paste a simple business input JSON.
- Validate against schema.
- Generate a basic financial model outline.
- Produce a verification checklist.
- Mark as approved or rejected.
